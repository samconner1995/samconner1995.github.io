Engine: GameMaker
Language: GML

A fast-paced hybrid of card battler and action game for up to 8 players!

My Role

Programmer + Game Designer

AI Behavior

Through use of steering vectors and finite state machines, dynamic behavior is achieved! Computer players look at their internal state and the environment around them, and calculate their desired movement vectors and actions in real time.

Online Play

Online play is achieved through the Steamworks API. The game state is synced using an authoritative server that validates requests from clients before sending approval.

Procedural Generation

Using weighted randomness, a diverse set of enemies and upgrade choices are generated. Players navigate through these choices in between battles. Each playthrough is different!

Data collection

Data is collected in the background in real time during battles. This is then processed and shown to the players after each battle. Players love validating their strategies through data!

Animation System

Each player's body is broken up into 6 pieces. These pieces are animated independently via custom animation code. Players' legs run, sidestep, and backpeddle. The force of firing a gun travels from the gun through the arms to the shoulders.

8 player input management

Up to 8 players can play locally. Keyboard and gamepads are supported. Using the OpenXInput API, 8 XINPUT devices can be used simultaneously, bypassing the API's limit of 4!

3d Card effects (UI)

Using surfaces, shaders, and some math, a 3d effect is created for the playing cards. Small UI touches like this go a long way!
